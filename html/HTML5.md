## Html5 shiv
    html5shiv 是一个针对 IE 浏览器的 HTML5 JavaScript 补丁，目的是让 IE 识别并支持 HTML5 元素。html5shiv 主要解决 HTML5 提出的新的元素不被 IE6-8 识别，这些新元素不能作为父节点包裹子元素，并且不能应用CSS样式。
    在条件判断里面使用。
```
<!--[if lt IE 9]>
  <script src="https://cdn.bootcss.com/html5shiv/r29/html5.js"></script>
<![endif]-->
```
